# Business Sales Analysis Dashboard
### Internship Task 1 - Data Science & Analysis

## Project Overview
This project involves performing exploratory data analysis on a retail sales dataset to identify revenue trends, top products, and regional performance using Power BI.

## Key Insights & Recommendations
Top Revenue Driver: The Technology category (specifically Copiers and Phones) consistently generates the highest revenue. 
Profitability Warning: While Furniture has high sales volume, the profit margins are low due to high shipping costs and heavy discounting. 
Regional Growth: The West Region is the most profitable.
Sales Trends: Revenue peaks significantly in Q4 (November-December). 

## Tools Used
Power BI: Data cleaning (Power Query), DAX Measures, and Visualization.
Dataset: Superstore Sales Dataset.
